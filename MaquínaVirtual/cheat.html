Maquinas virtuales


¿Qué es una maquina virtual?

Según la página oficial de IBM, uno de los precursores del pc como lo conocemos hoy en dia, define que: "Una máquina virtual (virtual machine) es una representación virtual o emulación de un equipo físico que emplea software en lugar de hardware para ejecutar programas e implementar aplicaciones".

Estas maquinas nos permiten correr en nuestros ordenadores software como sistemas operativos y todas las funcionalidades de este, para así realizar pruebas con los mismos, probar algún que otro sistema operativo u otra como utilizar dos sistemas operativos a la vez, como si tuvieramos un dual boot o doble arranque pero sin la instalación directa en nuestro disco.


¿Cómo operan estas maquinas?

Básicamente, este software se encarga de tomar los recursos de hardware de tu ordenador, como la ram, el almacenamiento el procesador y demás, para utilizarlos como si fuese su propio PC, utilizandp memorias virtuales, cache y potencia de ejecución para mover sus propios programas independientes a tu pc principal.


Problemas Comunes de las Virtual Machines

Aunque si bien se escucha muy buena la idea, hay que destaca que el sistema operativo que se va a ejecutar en tu maquina virtual, estará usandp tus recursos para funcionar, así que si no cuentas con un pc mínimamente potente para asignarle las especificaciones necesairas para arrancar este sistema(dependiendo del SO o sistema operativo que quieras arrancar será la demanda que este necesite) la experiencia con estas maquinas virtuales será un completo dolor de cabeza
